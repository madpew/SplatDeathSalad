SplatDeathSalad
===============
This is a modified version of SplatDeathSalad, a game by Sophie Houlden for the 7dfps-competition.

# further information: #
* Original Entry: http://7dfps.org/?projects=splat-death-salad
* http://www.sophiehoulden.com/
* http://www.7dfps.org/
